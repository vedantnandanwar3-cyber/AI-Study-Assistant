# AI Study Assistant

An AI-powered study assistant built using FastAPI, JavaScript, HTML, CSS, and SQLite.

## Features

- User Signup/Login
- Study Planner
- AI Chatbot
- DSA Tracker
- Dark/Light Theme
- Local Storage Support
- FastAPI Backend
- SQLite Database

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- FastAPI
- SQLite
